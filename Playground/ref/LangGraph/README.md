# LangGraph-in-action

LangGraph 实战课程代码库。

## 课程资源

本代码库对应 B 站课程：[LangGraph实战](https://www.bilibili.com/video/BV1hJqhBhE48?vd_source=9465c36dedefc94b04e58e4764092bef&spm_id_from=333.788.videopod.sections)

## 目录结构

*   **01入门与基础组件.ipynb**: LangGraph 基础概念与核心组件介绍。
*   **02状态管理与持久化执行.ipynb**: 深入了解状态管理机制及如何实现持久化。
*   **03流程控制与交互特性.ipynb**: 掌握复杂的流程控制与人机交互模式。
*   **04多智能体系统.ipynb**: 构建多 Agent 协作系统。
